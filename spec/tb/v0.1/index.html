Trust Broadcast Spec v0.1

A minimal verifiable broadcast layer for devices, drones, and nodes.

Structure

{
  "spec": "TB-0.1",
  "device": {...},
  "time": "...",
  "location": {...},
  "status": {...},
  "hash": "...",
  "signature": "..."
}

Principles

  • Self-contained
  • Verifiable
  • Non-repudiation

Verification

Clients must verify hash and signature before accepting any broadcast.
